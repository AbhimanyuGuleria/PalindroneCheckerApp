# Palindrome Checking App

## Objective

The objective of this project is to create an application that checks whether a given word, number, or sentence is a palindrome (a word that reads the same forward and backward).

## Project Procedure

1. The user enters a word or number.
2. The program converts the input to lowercase.
3. The input is reversed.
4. The original input is compared with the reversed input.
5. The result is displayed as **Palindrome** or **Not a Palindrome**.

## Use Case

* Helps beginners understand string operations.
* Useful for learning basic programming logic.
* Quick tool to check palindrome words or numbers.
